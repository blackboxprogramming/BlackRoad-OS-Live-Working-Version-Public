Moving items
============

Manual move
-----------
You can move files manually if needed, but keep the file metadata truthful.
That means updating:
- Layer
- Current Step
- Current Path
- Last Updated
- Source when crossing into a new layer

Scripted move
-------------
Use move-item.sh for safer movement.

Same layer example
------------------
./move-item.sh Canonical/01-Enterprise/Workflow/01-Intake/example.item.txt 02-Register

Cross-layer copy example
------------------------
./move-item.sh --copy Canonical/.../24-Conveyor-Belt/example.item.txt 01-Intake Canonical/.../13-Pull-Request/Workflow

Why copy sometimes
------------------
Copy mode is good when you want an audit trail in the upstream layer while also opening a live item in the downstream layer.
