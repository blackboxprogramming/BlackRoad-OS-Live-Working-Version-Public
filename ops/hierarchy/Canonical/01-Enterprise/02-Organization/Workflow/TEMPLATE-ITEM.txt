Template item
=============

Purpose
-------
Use this file as the starting pattern for any item that will move through this layer's 27-step workflow.
Copy this template, rename it, and keep the item's state aligned with the folder it currently lives in.

Notes by step
-------------
01-05: capture arrival, naming, category, and boundaries
06-10: break into atomic work and commit it to execution
11-16: gather context and produce the change or artifact
17-20: validate quality and make the decision to continue
21-25: package, stage, queue, and transfer downstream
26-27: record traceability and preserve long-term reference

Handoff note
------------
When the item reaches 24-Conveyor-Belt and then 25-Transfer, duplicate or move it into the next layer's Workflow/01-Intake/ and update both copies if you want an audit trail.
