## Post-Incorporation Checklist

### Immediate (Week 1)

- [ ] Verify Certificate of Incorporation filed with Delaware
- [ ] Confirm EIN received from IRS
- [ ] Review and sign all incorporation documents in Atlas Dashboard
- [ ] File 83(b) election within 30 days of stock grant
- [ ] Set up corporate bank account
- [ ] Activate Stripe payment processing

### Short-Term (Month 1)

- [ ] Set up corporate accounting (QuickBooks, Xero, or similar)
- [ ] Register as a foreign corporation in operating state(s) if different from Delaware
- [ ] Obtain business licenses as required
- [ ] Set up payroll system
- [ ] Review and customize Bylaws if needed
- [ ] Document initial capitalization table (cap table)

### Ongoing Compliance

- [ ] Annual Delaware Franchise Tax filing (due March 1)
- [ ] Annual registered agent renewal ($100/year after first year)
- [ ] Annual report filing with Delaware
- [ ] Board meeting minutes (at least annually)
- [ ] Maintain corporate records book
- [ ] File federal and state tax returns
