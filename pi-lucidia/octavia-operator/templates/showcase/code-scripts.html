✨ The 10 Golden Rules
Every code and script template follows these rules:
1.Always include comments
2.Commands must work
3.Use full paths
4.Group related actions
5.Include verification
6.Escape properly
7.One block = one task
8.Comments start with #
9.Blank lines separate
10.End with verification
