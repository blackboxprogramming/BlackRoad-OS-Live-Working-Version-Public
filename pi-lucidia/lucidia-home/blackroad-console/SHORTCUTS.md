# BlackRoad OS Keyboard Shortcuts

## Window Navigation

| Shortcut | Action |
|----------|--------|
| `Alt + Tab` | Switch to next window |
| `Alt + Shift + Tab` | Switch to previous window |

## Quick Panel Access

| Shortcut | Panel | Icon |
|----------|-------|------|
| `Alt + 1` | Terminal | 🖥️ |
| `Alt + 2` | Agent Registry | 🤖 |
| `Alt + 3` | Health Monitor | ❤️ |
| `Alt + 4` | Memory Vault | 💎 |
| `Alt + 5` | Token Vault | 🔐 |
| `Alt + 6` | Agent Builder | 🏗️ |
| `Alt + 7` | Billing & Revenue | 💰 |
| `Alt + 8` | File Manager | 📁 |
| `Alt + 9` | Settings | ⚙️ |

## Window Actions

| Action | Method |
|--------|--------|
| Move Window | Drag window header |
| Resize Window | Drag bottom-right corner |
| Minimize | Click yellow button (–) |
| Maximize/Restore | Click green button (⬜) |
| Close | Click red button (×) |
| Focus Window | Click anywhere on window |

## Dock Shortcuts

| Action | Method |
|--------|--------|
| Open Panel | Click dock icon |
| Restore Minimized | Click dock icon of minimized window |
| Show Active | Blue dot under dock icon |

## Special Features

| Action | Method |
|--------|--------|
| Operator Center | Click "🎯 OPERATOR" button (top-right) |
| Current Time | Displayed in top bar (24-hour format) |

---

**Pro Tip:** All shortcuts use `Alt` as the modifier key to avoid conflicts with browser shortcuts.
