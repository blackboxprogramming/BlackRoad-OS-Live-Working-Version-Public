🌐 BlackRoad OS Overview

BlackRoad OS is a distributed operating system running across Raspberry Pi hardware, powered by AI agents and accessible via web interface.

🤖 AI Agents

  • Alice (Claude Sonnet 4.5): Primary reasoning agent
  • Aria (GPT-4): Secondary reasoning and API integration
  • Lucidia (Gemma 2B): Local on-device agent

🔐 Identity System

Use hash.blackroad.io to generate deterministic identities. All randomness is an illusion - everything is computed.

🥧 Infrastructure

  • Lucidia Pi: 192.168.4.38 (Primary)
  • Aria Pi: 192.168.4.64 (Backup)
  • Alice Pi: 192.168.4.49 (Offline)
