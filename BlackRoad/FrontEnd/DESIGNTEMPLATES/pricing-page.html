Frequently asked questions

Why is this free? −
BlackRoad OS is fully self-hosted on hardware you own. There's no SaaS fee because there's no SaaS. You run everything — the code, the infrastructure, the AI models. The only ongoing costs are electricity and optional cloud VPS.
What hardware do I need? +
Minimum: one Raspberry Pi 5 (4GB+) with a 32GB microSD card. Recommended: Pi 5 (8GB) with a Hailo-8 accelerator for AI inference. Our full fleet uses 5 Pi 5 boards with 2 Hailo-8 units (52 TOPS total).
What does $607/mo in SaaS replace? +
n8n ($50), Temporal ($200), Airbyte ($300), Datadog ($20), Vercel Pro ($20), GitHub Team ($17). All replaced by self-hosted alternatives: Gitea, Docker Swarm, cron automation, and Cloudflare free tier.
Can I start with one node and scale up? +
Yes. Start with a single Pi, install BlackRoad OS, and add nodes to the WireGuard mesh as you grow. Each new node auto-joins the cluster and gets a RoadNet subnet.
