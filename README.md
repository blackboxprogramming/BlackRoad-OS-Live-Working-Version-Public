# BlackRoad OS -- Live Working Version (Public)

This repository is the public-facing live/demo surface for BlackRoad OS.

## Role

`BlackRoad-OS-Live-Working-Version-Public` should host the polished public version of the live working experience. It is the public projection of the private live proving ground and should only contain routes, pages, and demo flows that are safe, intentional, and working.

## What Belongs Here

- polished public app shells and demo experiences
- public-facing routes that are already validated privately
- curated public-safe integrations and examples
- production-safe marketing and product surfaces backed by real rendering

## Shared Four-Repo Split

- `BlackRoad-OS-Live-Monorepo-Private` = canonical brain
- `BlackRoad-OS-Live-Working-Version-Private` = live proving ground
- `BlackRoad-OS-Live-Monorepo-Public` = curated public mirror
- `BlackRoad-OS-Live-Working-Version-Public` = polished public live/demo surface

## Publishing Rule

Only promote routes and experiences here after they work in the private live repo and can be exposed safely.

## Immediate Priorities

1. Keep the public live surface polished and intentional.
2. Expose only public-safe app routes and demos.
3. Mirror working experiences from the private live repo after validation.
4. Avoid placeholder pages, dead links, and incomplete operator-only flows.
