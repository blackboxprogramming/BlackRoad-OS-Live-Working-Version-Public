02 — Quickstart

Make your first request

Get your API key from CarKeys, then call any endpoint. Every SDK wraps the same REST layer.

BASH
Run in playground ↗ 
# Get your API key from CarKeys → carkeys.blackroad.io

curl https://api.blackroad.io/v1/agents \
  -H "Authorization: Bearer $BLACKROAD_API_KEY" \
  -H "Content-Type: application/json" \
  -H "X-BlackRoad-Version: 2026-04-11"
JAVASCRIPT / TYPESCRIPT
Run in playground ↗ 
import { BlackRoad } from '@blackroad/sdk'

const br = new BlackRoad({
  apiKey: process.env.BLACKROAD_API_KEY,
})

// List all agents on the platform
const agents = await br.agents.list()
console.log(agents.data)

// Post an event
await br.events.publish({
  type:    'agent.invoked',
  agentId: 'road',
})
PYTHON
Run in playground ↗ 
import os
import blackroad

# Initialise with your API key from CarKeys
br = blackroad.Client(
    api_key=os.environ["BLACKROAD_API_KEY"]
)

# List all agents
agents = br.agents.list()
for agent in agents.data:
    print(agent.id, agent.status)

# Post an event
br.events.publish(type_="agent.invoked", agent_id="road")
03 — Authentication

Bearer token auth

All API requests require a Bearer token issued by CarKeys. Tokens are scoped, rotatable, and audit-logged on every request.

Required headers
Header Value Required
Authorization Bearer <token> — your CarKeys API key Yes
Content-Type application/json Yes
X-BlackRoad-Version 2026-04-11 — pin the API schema version Recommended
Error codes
Code Status Description
401 Unauthorized Missing or invalid Bearer token. Re-issue from CarKeys.
403 Forbidden Token is valid but lacks the required scope for this endpoint.
429 Rate Limited Request rate exceeded. Retry after the Retry-After header value.
500 Server Error Unexpected platform error. Check status.blackroad.io and retry.
Standard plan 1,000 req/min
Enterprise plan 10,000 req/min
Burst window 60s

Rate limit headers on every response:
X-RateLimit-Limit · X-RateLimit-Remaining · X-RateLimit-Reset
