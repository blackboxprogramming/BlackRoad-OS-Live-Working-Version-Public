05 — Rules
Design Laws
Non-negotiable. Every app, every component, every agent that builds UI must follow these.
✓ Do
✓ Text is white (#f5f5f5) or black — always
✓ Backgrounds: #000, #0a0a0a, #111 — only these three levels
✓ Shapes with color → use the full gradient
✓ Gradient bars, borders, icons, orbs, shapes = full color
✓ Buttons: gradient fill OR white fill OR outlined
✓ Cards: #0a0a0a bg, 1px #1a1a1a border, 10px radius
✓ Fonts: Space Grotesk + JetBrains Mono
✓ Border radius: cards 10px · buttons 6px · inputs 6px · badges 4px
✕ Never
✕ Colored text — no #FF2255, #4488FF, or any brand hex on text
✕ Solid brand colors as button/card backgrounds
✕ Light or bright backgrounds (#fff, #f5f5f5, #dcfce7, etc.)
✕ Off-brand dark navies (#1a1a2e, #12122a, #0a0a0a)
✕ Platform/status colors as text (#22c55e green, #ef4444 red, etc.)
✕ Gradient on body text or UI labels
✕ More than 3 background levels on one page
✕ Any font other than Space Grotesk / JetBrains Mono / Inter
06 — Tokens
CSS Variables
Copy these into any worker, page, or component. The full system in one :root block.
/* ── Backgrounds ── */
--bg: #000;
--card: #0a0a0a;
--elevated: #111;
--hover: #181818;
--border: #1a1a1a;
 
/* ── Text ── */
--text: #f5f5f5;
--sub: #737373;
--muted: #444;
 
/* ── Gradient ── */
--g: linear-gradient(
  90deg,
  #FF6B2B, #FF2255,
  #CC00AA, #8844FF,
  #4488FF, #00D4FF
);
 
/* ── Fonts ── */
--sg: 'Space Grotesk';
--jb: 'JetBrains Mono';
--in: 'Inter';
 
/* ── Radius ── */
--r-sm: 4px; /* badge */
--r: 6px; /* button, input */
--r-md: 8px; /* icon */
--r-lg: 10px; /* card, nav */
