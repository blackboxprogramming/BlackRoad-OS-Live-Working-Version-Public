02 — API

API Reference

REST API for the BlackRoad platform. Base URL: https://api.blackroad.io

Endpoints
GET /v1/agents
POST /v1/auth
GET /v1/products
GET /v1/events
GET /v1/users
GET /v1/orgs
Request 
curl https://api.blackroad.io/v1/agents \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json"
Response 200 OK application/json 
{
  "data": [
    {
      "id": "agent_01HXKP93BZQM",
      "name": "road",
      "status": "active",
      "model": "claude-sonnet-4.5",
      "capabilities": ["chat", "code", "search"],
      "created_at": "2026-01-14T09:24:00Z"
    }
  ],
  "meta": {
    "total": 69,
    "page": 1,
    "per_page": 20
  }
}
List agents
GET https://api.blackroad.io/v1/agents
Query parameters
limit integer Max results (default 20, max 100)
status string Filter by active · idle · offline
org_id string Scope to specific organization
Authorization header required Bearer token from CarKeys
