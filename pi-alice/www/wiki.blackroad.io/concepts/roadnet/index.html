RoadNet
The Carrier Network
A WiFi mesh network across all 5 Pis. SSID: RoadNet. Password: BlackRoad2026. Backup connectivity when WireGuard is down.
The Mesh
Alice — Channel 1. Pi 400. Main gateway, 48+ domains, Pi-hole DNS.
Cecilia — Channel 6. Pi 5. CECE API, TTS, MinIO, Hailo-8 accelerator.
Octavia — Channel 11. Pi 5. 1TB NVMe, Gitea, Docker Swarm leader.
Aria — Channel 1. Pi 5. Portainer, Headscale, creative workloads.
Lucidia — Channel 11. Pi 5. GitHub Actions runner, 334 web apps.

Each Pi broadcasts a WiFi access point. Five APs, three channels, zero overlap. Full mesh coverage.

Subnets

Each node gets its own /24 subnet: 10.10.1.0/24 through 10.10.5.0/24. NAT through wlan0 for internet access. Pi-hole handles DNS. WireGuard failover kicks in automatically if the primary tunnel drops — traffic reroutes through the mesh.

Boot Persistence

Two systemd units: roadnet.service brings up the access point on boot. roadnet-failover.service monitors the WireGuard tunnel and reroutes if it drops. Both enabled, both persistent. Survives reboots, power cycles, and SD card corruption.

Deployment

Scripts at ~/roadnet/ — deploy, status, uninstall, node-setup, boot. The master topology lives in WIRE-MAP.md. One command deploys to all nodes. One command checks the entire mesh.

Why

Redundancy. If the home router dies, if WireGuard drops, if Cloudflare has an outage — the Pis can still talk to each other over RoadNet. The mesh is the last line of defense. When everything else fails, the local network survives.
