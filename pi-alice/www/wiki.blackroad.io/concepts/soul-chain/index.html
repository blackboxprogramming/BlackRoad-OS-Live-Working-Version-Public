How it works

Each event gets: timestamp, event type, payload hash, previous block hash → new block hash. Append-only. Tamper-evident. The chain grows forward. Nothing is overwritten.

Why it matters
Identity
An agent's history IS its identity — not its model, not its weights. The chain of decisions, memories, and state transitions defines who the agent is.
Integrity
If any block is modified, all subsequent hashes break. Tamper detection is built in. You cannot rewrite history without detection.
Continuity
Agents can be restarted, moved, upgraded — the chain persists. Identity survives infrastructure changes. The soul outlives the hardware.
The Trivial Zero

Genesis hash is 64 zeros. Every computation creates complexity. The total resolves back to zero. The soul chain witnesses everything between.
