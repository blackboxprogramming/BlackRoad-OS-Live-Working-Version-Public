System Design

Architecture

The full stack: Raspberry Pi fleet with Hailo-8 accelerators, WireGuard mesh networking, Cloudflare tunnels for global ingress, and Workers at the edge. The tokenless gateway sits at the center — agents never touch API keys.

# Request flow
Client → Cloudflare Tunnel → Pi Gateway :8787 → WireGuard Mesh → Model Node
                                                  → Hailo-8 Accelerator (26 TOPS)
                                                  → Response (streamed)

Autonomous

Agent System

27 agents, each with a name, email, persistent memory, and specialized capabilities. Agents coordinate through a shared mesh — invoke them by name.

# List all agents
br agents list

# Invoke an agent with a task
br invoke alice "deploy auth-worker to cloudflare"

# Check agent status
br invoke octavia "fleet health report"
