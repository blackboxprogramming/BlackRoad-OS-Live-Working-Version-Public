Why SQLite
›
Zero Configuration
No server process. No port. No auth setup. Just a file. Copy it, back it up, query it. Simplest possible persistence.
›
FTS5
Full-text search built in. Ranked results. Prefix matching. Boolean operators. No Elasticsearch needed.
›
Portable
Every database is a single file. scp it anywhere. Works on every OS. No dependency hell.
›
HMAC Chain
Journal entries chained with HMAC. If anyone tampers with a record, the chain breaks. Cryptographic integrity.
