Architecture
›
Physical Layer
5 WiFi access points. Each Pi node has its own subnet. Wired backbone between nodes. Pi-hole DNS on Alice.
›
WireGuard Mesh
Full mesh VPN. Every node can reach every other node directly. Internet → Gematria → WG → Pi fleet.
›
Headscale
Self-hosted WireGuard coordination on Aria. Peer discovery. Key exchange. Replaces Tailscale control plane.
›
NATS Messaging
CarPool (forked NATS v2.12.3) on Octavia. Pub/sub for agent coordination, fleet events, real-time sync.
›
Tor Overlay
Hidden services on 3 nodes. .onion addresses for global reachability without exposing public IPs.
›
DNS
PowerDNS on Lucidia + Gematria. Authoritative for all 19 domains. Pi-hole for local resolution + ad blocking.
