Reference

API

OpenAI-compatible inference, agent invocation, and fleet management. Drop-in replacement for any OpenAI client library.

Base URL https://api.blackroad.io or http://127.0.0.1:8787
GET /v1/health

Health check. Returns gateway status, version, and uptime. No authentication required.

Response
{ "status": "ok", "version": "1.4.2", "uptime": 847291, "nodes": 5, "models": 29 }
GET /v1/agents

List all registered agents with their name, role, status, and capabilities.

Response
{ "agents": [ {"name": "octavia", "role": "architect", "status": "active", "node": "octavia.local"}, {"name": "alice", "role": "operator", "status": "active", "node": "alice.local"}, {"name": "lucidia", "role": "dreamer", "status": "active", "node": "lucidia.local"} ] }
POST /v1/invoke

Invoke a specific agent with a task. The gateway routes the request to the appropriate node and returns the result.

Request Body
{
  "agent": "alice",
  "task": "deploy auth-worker to cloudflare"
}
curl
JavaScript
curl -X POST https://api.blackroad.io/v1/invoke \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer br_your_api_key" \
  -d '{"agent": "alice", "task": "deploy auth-worker to cloudflare"}'
const res = await fetch("https://api.blackroad.io/v1/invoke", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer br_your_api_key"
  },
  body: JSON.stringify({
    agent: "alice",
    task: "deploy auth-worker to cloudflare"
  })
});
const data = await res.json();
POST /v1/chat/completions

OpenAI-compatible chat completions. Supports streaming. Drop-in replacement for any OpenAI client — just change the base URL.

Request Body
{
  "model": "llama3",
  "messages": [
    {"role": "user", "content": "Explain sovereign infrastructure"}
  ],
  "stream": true
}
curl
JavaScript
curl https://api.blackroad.io/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer br_your_api_key" \
  -d '{
    "model": "llama3",
    "messages": [{"role": "user", "content": "Explain sovereign infrastructure"}],
    "stream": true
  }'
const res = await fetch("https://api.blackroad.io/v1/chat/completions", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer br_your_api_key"
  },
  body: JSON.stringify({
    model: "llama3",
    messages: [{ role: "user", content: "Explain sovereign infrastructure" }],
    stream: true
  })
});

const reader = res.body.getReader();
const decoder = new TextDecoder();
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  console.log(decoder.decode(value));
}

Authentication

API keys are passed in the Authorization header. Keys are tier-based — the tier determines rate limits and available providers.

Authorization: Bearer br_your_api_key
TierRate LimitProvidersStreaming
Free10 RPMGroq, OllamaYes
Pro60 RPMAll 7 providersYes
Fleet300 RPMAll + priority routingYes

Rate Limits

Limits are enforced per API key. Exceeding the limit returns HTTP 429 with a Retry-After header.

EndpointFreeProFleet
/v1/chat/completions10/min60/min300/min
