Guide

Get Started

From zero to running AI infrastructure in under five minutes. No cloud accounts required.

1

Install

One command installs the br CLI, configures your local environment, and connects to the BlackRoad gateway.

curl -fsSL https://docs.blackroad.io.sh | bash
Installing BlackRoad CLI v1.4.2... Detected: macOS arm64 Binary: /usr/local/bin/br Config: ~/.config/blackroad/ Done. Run `br status` to verify.
2

Check Status

Verify the installation and see what the fleet looks like. Shows gateway health, active nodes, available models, and agent count.

br status
Gateway: https://api.blackroad.io [ok] Nodes: 5 online (alice, cecilia, octavia, aria, lucidia) Models: 29 loaded across 7 providers Agents: 14 registered, 12 active Compute: 26 TOPS (2x Hailo-8)
3

Talk to AI

Chat with any model through the CLI. Requests route through the mesh to the fastest available node. Streaming by default.

br chat "explain quantum computing in one paragraph"
Quantum computing uses qubits that can exist in superposition — simultaneously 0 and 1 — enabling parallel exploration of solution spaces. Through entanglement and interference, quantum algorithms solve certain problems exponentially faster than classical machines... [llama3 via cecilia, 847ms first token]
4

List Agents

See all 14 agents with their role, status, and assigned node. Each agent has specialized capabilities and persistent memory.

br agents list
NAME ROLE STATUS NODE octavia architect active octavia.local alice operator active alice.local lucidia dreamer active lucidia.local aria interface active aria.local cecilia analyst active cecilia.local shellfish hacker idle — ...
5

Invoke an Agent

Give a task to a specific agent. The agent uses its tools, memory, and capabilities to complete the work, then reports back.

br invoke alice "deploy my-worker to cloudflare"
Routing to alice@alice.local... Task: deploy my-worker to cloudflare Status: Building worker from ./src/index.ts Status: Deploying to production Done: https://my-worker.blackroad.workers.dev Latency: 12.4s
6

Search the Database

Full-text search across all indexed infrastructure — repos, scripts, agents, configs. FTS5-powered, sub-millisecond results.

br search "fleet health"
3 results for "fleet health": 1. health-check (tool) Fleet health monitoring and alerting 2. fleet-health-report.sh (script) Daily fleet status aggregator 3. kpi-fleet-collector.sh (script) KPI collection for fleet metrics
7

Join the Fleet

Connect your own machine as a node in the mesh. Contributes compute, gets access to all models and agents across the network.

br fleet join
Generating WireGuard keypair... Registering node: your-hostname Assigned IP: 10.8.0.12 Tunnel: active You are now part of the BlackRoad mesh.
