API Reference

BlackRoad OS API

12 endpoints. REST + JSON. API key in header.

curl -H "Authorization: Bearer br_live_YOUR_KEY" \ https://blackroad.io/api/health
Authentication

All endpoints except /api/health require an API key. Get one at app.blackroad.io/signup.

Endpoints
GET/api/healthpublicService status

Returns service health, agent count, and version.

{"status":"alive","agents":69,"nlp":true,"host":"alice"}
GET/api/agentspublicList all agents

Returns all 69 registered agents with roles, groups, and status.

[{"id":"alice","name":"Alice","role":"Gateway","group":"fleet","ip":"192.168.4.49"}, ...]
POST/api/chatapi keyChat with agent

Send a message to any agent. Returns AI-generated response via local Ollama.

// Request {"agent":"alice","message":"what is your status?","channel":"general"} // Response {"agent":"alice","name":"Alice","reply":"All systems operational. 37 sites served, DNS filtering active."}
POST/api/actionapi keyExecute script

Execute any of 231 actions across 71 intents. Real scripts run on the Pi fleet.

// Request {"intent":"search","query":"fleet health"} // Response {"intent":"search","action":"Codex search","status":"ok","output":"..."}
POST/api/smartapi keyNLP + action + AI

Send natural language. Parses intent, executes matching script, and returns AI response.

// Request {"message":"search codex for fleet health"} // Response {"parsed":{"intents":[{"intent":"search"}]},"action":{"status":"ok","output":"..."},"reply":{"agent":"road","text":"..."}}
GET/api/actionspublicList all actions

Returns all 71 intents and 231 available actions.

{"total_intents":71,"total_actions":231,"actions":{"status":["Fleet status","Heartbeat"],...}}
GET/api/fleetapi keyLive fleet data

Returns live fleet node status, agent counts, and RoundTrip health.

GET/api/productspublicProduct list

Returns product catalog for the current domain.

GET/api/messagesapi keyChat history

Returns message history for a channel. Query: ?channel=general&limit=50

GET/api/channelspublicList channels

Returns all 8 chat channels: general, fleet, iot, security, creative, ops, research, ceo.

GET/api/groupspublicAgent groups

Returns agent groups: fleet, cloud, services, ai, ops, myth, lead, iot.
