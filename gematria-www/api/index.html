API Documentation

RESTful endpoints for every service in the fleet. No auth required for read access.

Base URL
https://backend.blackroad.io
Endpoints
GET/health
Health check. Returns fleet uptime and node count.
statusuptimenodes
GET/api/stats
Aggregate statistics across the entire fleet.
domainsorgsreposagentsnodes
GET/api/domains
List all 19 registered domains and their DNS status.
domainregistrarstatusdns_provider
GET/api/orgs
List all 16 GitHub organizations and Gitea orgs.
nameplatformrepo_counturl
GET/api/repos
List all 254 repositories across all orgs.
nameorgplatformlanguageupdated
GET/api/agents
List all 67 active agents and their assigned nodes.

GET/api/nodes
Detailed info for each of the 7 fleet nodes.

Quick Start
curl example
# Get fleet statistics
curl -s https://backend.blackroad.io/api/stats | python3 -m json.tool
Example Response — /api/stats
{
  "domains": 19,
  "orgs": 16,
  "repos": 254,
  "agents": 67,
  "nodes": 7,
  "tops": 52,
  "monthly_cost": "$38",
  "uptime": "99.2%"
}
fetch example
const res = await fetch('https://backend.blackroad.io/api/agents');
const agents = await res.json();
console.log(`${agents.length} agents online`);
All endpoints return JSON. No API key required for read access.
