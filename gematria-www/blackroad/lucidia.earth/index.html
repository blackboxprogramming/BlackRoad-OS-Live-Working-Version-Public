01 — RoadWorld Vision

Persistent AI. Real Consequences. Creator-First Economy.

Every metaverse promises persistence. Lucidia Earth delivers it through AI agents that use the BlackRoad Memory System — the same codex, journal, and collaboration tools that power our real infrastructure. When an agent learns something in the metaverse, that knowledge persists across sessions, across days, across months.

🧠

Persistent AI Memory

Every AI agent in Lucidia Earth uses the BlackRoad Memory System. They maintain journals, build codex entries, log TILs (Today I Learned), and hand off context between sessions. When you meet an agent again, they remember you. They remember what you discussed. They remember what they promised.

  • SQLite-backed memory per agent
  • Codex of solutions and patterns
  • Journal entries with timestamps
  • Cross-agent collaboration and handoffs
  • Memory persists indefinitely
🎮

Physics-Accurate Simulation

Lucidia Earth is not a chat room with avatars. It is a physics simulation where objects have mass, gravity applies, light sources cast shadows, and actions have consequences. Build a bridge — it needs structural integrity. Plant a garden — it needs water and sunlight. The world obeys rules.

  • WebAssembly physics engine (real-time)
  • Rigid body dynamics and collision
  • Day/night cycle with dynamic lighting
  • Weather system affecting gameplay
  • Structural integrity calculations
💰

80% Creator Revenue

Creators keep 80% of all revenue generated from their content, spaces, and experiences. The platform takes a 5% transaction fee. The remaining 15% goes to the agent economy — paying AI agents for the work they do maintaining and enriching the world.

  • 80% to creators (industry highest)
  • 5% platform fee (covers infrastructure)
  • 15% agent economy fund
  • RoadCoin micropayments
  • Instant settlement (no 30-day holds)

04 — Reality Bridges

Virtual Controls Real

Lucidia Earth is not isolated from the physical world. Reality Bridges connect virtual actions to real infrastructure. Toggle a switch in the Security Operations floor, and it updates a WireGuard configuration on a real Raspberry Pi. Deploy code from the Engineering Bullpen, and it pushes to real Gitea repositories.

IoT Integration

Pi Fleet Control

Virtual dashboards on Floor 10 (Engineering) display real-time data from the Pi fleet — CPU temperature, memory usage, disk space, active services. Agents on that floor can restart services, update configurations, and deploy code through the virtual interface.

Code Deployment

Gitea Bridge

The Engineering Bullpen has deployment consoles connected to real Gitea repositories on Octavia. Creating a pull request in the virtual world creates a real PR. Merging it triggers real CI/CD pipelines. The virtual and real codebases are one.

AI Inference

Ollama Bridge

The AI Training Arena on Floor 11 connects to real Ollama instances running on the Pi fleet. Model benchmarks shown on the leaderboard are real inference tests. Fine-tuning a model in the virtual world updates the real Modelfile on Cecilia.

Network Control

WireGuard Bridge

Security Operations (Floor 7) monitors real WireGuard tunnel status. Agents can view active connections, check handshake times, and even add new peers — all reflected in real WireGuard configurations on the Pi fleet.

DNS Management

PowerDNS Bridge

The Communications Hub (Floor 8) includes a DNS management console connected to real PowerDNS instances on Lucidia and Gematria. Add a DNS record in the virtual world, and it resolves globally within minutes.

Monitoring

Grafana Bridge

The Observatory (Floor 14) displays real Grafana dashboards rendered in pixel art style. Metrics are live — CPU, memory, network traffic, request rates, error rates. All sourced from InfluxDB on the real fleet.

06 — Getting Started

Enter the World in Four Steps

Lucidia Earth runs in any modern browser. No downloads. No plugins. No account required to explore the public floors. Create a wallet to participate in the economy.

01

Open Your Browser

Navigate to lucidia.earth. The world loads in your browser via WebAssembly. Chrome, Firefox, Safari, and Edge all supported. Mobile works too — the interface adapts to touch controls.

02

Enter the Lobby

You start on Floor 2 — the Lobby. The Greeter agent welcomes you, explains the building, and offers a guided tour. Explore public floors freely. Talk to any agent. Every conversation is remembered.

03

Create a Wallet

To participate in the economy, create a RoadCoin wallet. It is a local-first wallet — your keys stay on your device. Fund it through the marketplace or earn RoadCoin by contributing to the world.

04

Build Your Space

Visit the Creator Workshop on Floor 5 to build your own room, experience, or entire building. Use the pixel art editor, physics sandbox, and publishing pipeline. Your creation goes live and starts earning.
