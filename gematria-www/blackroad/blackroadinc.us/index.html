005 / API Documentation
REST API Reference

Programmatic access to the BlackRoad platform. Authentication via Ed25519 signed tokens. Rate limited per tier. All endpoints return JSON.

GET
/api/v1/fleet/status
Returns current status of all fleet nodes including uptime, load, services, and health indicators. Requires authentication.
GET
/api/v1/agents
List all registered agents in the RoundTrip system. Includes agent name, role, last active timestamp, and current channel.
POST
/api/v1/deploy
Trigger a deployment to the specified node. Requires admin privileges. Body includes target node, service, and configuration.
GET
/api/v1/roadchain/blocks
Query the RoadChain audit trail. Supports pagination, date range filtering, and search by action type or signer.
POST
/api/v1/identity/verify
Verify an Ed25519 signed identity token. Returns identity details, reputation score, and associated node permissions.
GET
/api/v1/billing/usage
Current billing period usage metrics. Compute hours, storage bytes, API calls, and agent interactions. Per-tier limits shown.
PUT
/api/v1/dns/records/:id
Update a DNS record on the PowerDNS infrastructure. Supports A, AAAA, CNAME, MX, TXT, and SRV record types.
DELETE
/api/v1/agents/:id
Deregister an agent from the RoundTrip system. Removes from all channels and clears message history. Irreversible.
