Architecture
›
Lexer
Tokenizer with Python-style indentation — colon triggers INDENT/DEDENT. Handles strings, integers, floats, operators, keywords.
›
Parser
Full AST generation from token stream. Recursive descent parser. Supports nested scopes, expressions, control flow.
›
Interpreter
Tree-walking evaluator. Executes AST nodes directly. Environment-based scoping with closures.
Features
›
Functions & Recursion
First-class functions. Closures. Recursive calls. let/var/const variable declarations.
›
Control Flow
if/elif/else, while loops, for loops. Break and continue. Match expressions (pattern matching).
›
Types
Strings, integers, floats, booleans, arrays, objects. Dynamic typing with runtime checks.
›
Concurrency
spawn keyword for launching concurrent tasks. Channel-based communication between spawned processes.
›
3D Primitives
space keyword for creating 3D primitives — cubes, spheres, planes. Built for the RoadWorld metaverse engine.
›
Python-Style Syntax
Colon-based blocks, significant indentation. Familiar to Python developers but with unique extensions.
