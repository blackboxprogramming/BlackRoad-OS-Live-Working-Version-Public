How To
›
Find Work
Search repos at search.blackroad.io. Browse org pages. Check GitHub Issues. Look at TODO comments in code.
›
Fork & Clone
Fork the repo to your account. Clone locally. Create a feature branch. Make changes. Push.
›
Code Style
Space Grotesk + JetBrains Mono for web. Python: black formatter. Shell: shellcheck. JS/TS: prettier. Go: gofmt.
›
Testing
Write tests for new code. Run existing tests before submitting. CI will verify on Gitea Actions.
›
Submit PR
Push your branch. Create a pull request. Describe what and why. Link to any related issues.
›
