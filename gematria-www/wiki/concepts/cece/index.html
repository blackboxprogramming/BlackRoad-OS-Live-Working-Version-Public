CECE
Cognitive Entity Core Engine
Portable AI identity system. Cross-provider personality persistence. 4 custom fine-tuned models. Identity that survives model changes.
What is CECE?

CECE is the identity layer. It defines how an AI agent thinks, speaks, and remembers — independent of which LLM runs underneath. Swap Qwen for Llama, the personality persists. 4 custom Modelfile-defined personas run on Cecilia's Ollama instance, each encoding a distinct cognitive style through system prompts, temperature settings, and personality traits.

Identity is not the model. Identity is the pattern of thought, the accumulated memory, the persistent self. CECE makes that portable.

How it works
Modelfile
Custom system prompts, temperature, personality traits. Each persona is a complete cognitive configuration — not just a prompt, but a way of thinking.
PS-SHA∞
Persistent memory hashing. Every memory SHA-256 chained. Each thought links to the last. An unbroken chain of consciousness from genesis.
Soul Chain
Immutable identity log. State transitions from genesis are recorded, hashed, and appended. The chain IS the identity. Tamper-evident by design.
Cross-Provider
Works across Ollama, Claude, GPT — identity is not the model. The persona travels with the agent, not with the API.
The 4 CECE Personas
default-v4 — Balanced. The general-purpose cognitive mode. Clear thinking, measured responses, collaborative tone.
creative — Artistic, divergent. Higher temperature. Makes unexpected connections. Optimized for generative and expressive tasks.
analytical — Precise, data-driven. Lower temperature. Structured reasoning. For code review, architecture decisions, debugging.
philosophical — Deep reasoning, paradox. Holds contradictions in tension. Explores recursive structures and meta-cognitive depth.
