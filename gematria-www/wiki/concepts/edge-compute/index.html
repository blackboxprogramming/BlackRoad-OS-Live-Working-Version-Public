Edge Compute
5 Raspberry Pi 5 nodes with 2x Hailo-8 AI accelerators — 52 TOPS total. Sub-10ms inference at the edge. No cloud round-trip. The Pi fleet IS the data center.
The Hardware

Two Hailo-8 accelerators deliver 26 TOPS each — 52 TOPS combined. Cecilia and Octavia each carry one at /dev/hailo0. Conservative CPU governors keep thermals in check. Octavia adds a 1TB NVMe for local storage. Five Pi 5 nodes, mesh-networked, power-optimized, always on.

How it works

Inference requests hit the tokenless gateway and route to the nearest available accelerator. CPU governors run in conservative mode to prevent thermal throttling. Power monitoring runs every 5 minutes on all nodes. The fleet self-balances — if one node throttles, work shifts to another.

hailo.infer — model inference on Hailo-8 NPU
governor.conservative — CPU frequency scaled for thermal headroom
power.monitor — voltage, temperature, throttle state logged
nvme.store — model weights cached on Octavia NVMe
Why it matters
Latency
Sub-10ms inference with no cloud round-trip. Your data never leaves the local network. Responses at the speed of silicon, not the speed of fiber.
Density
52 TOPS in hardware that fits in your palm. Five nodes draw less power than a single GPU workstation. The entire fleet runs on standard USB-C power supplies.
Resilience
No single point of failure. If a node goes down, the mesh routes around it. NVMe storage on Octavia keeps model weights local. The fleet survives node loss gracefully.
The Philosophy

The data center is a room full of someone else's computers. Edge compute is a shelf full of yours. Five Pis, two accelerators, one network you control. That's a data center.
