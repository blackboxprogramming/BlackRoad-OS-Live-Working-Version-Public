Self-Healing

Autonomy scripts run on every node. Heartbeat every minute. Heal every 5 minutes. Watchdog every 30 seconds. Services auto-restart on failure. Stats-proxy monitors everything. The fleet takes care of itself.

Heartbeat (1m)
Every node pings in every minute. If a heartbeat is missed, the fleet knows.
Heal (5m)
Checks critical services. Auto-restarts stats-proxy, Ollama, and other core services if down.

Alice runs task_watchdog.py on a 30-second timer. Redis task queue monitoring. Nothing slips through.
