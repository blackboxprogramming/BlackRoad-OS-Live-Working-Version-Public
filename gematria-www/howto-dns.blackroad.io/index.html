How to Self-Host DNS
Replace Cloudflare DNS with PowerDNS on a Pi

Steps
›
1. Install
sudo apt install pdns-server pdns-backend-sqlite3 — PowerDNS with SQLite backend. Lightweight, perfect for Pi.
›
2. Configure
Edit /etc/powerdns/pdns.conf. Set backend to gsqlite3. Create the schema. Add your zone files.
›
3. Add Records
pdnsutil create-zone yourdomain.com. pdnsutil add-record yourdomain.com @ A 1.2.3.4. Standard DNS record management.
›
4. Point NS Records
At your registrar (GoDaddy), change nameservers to your PowerDNS server IPs. Propagation takes 24-48h.
›
5. Redundancy
Run PowerDNS on two nodes (like Lucidia + Gematria). Zone transfers keep them in sync. Two NS records = redundancy.
›
6. Why
Cloudflare sees every DNS query your users make. PowerDNS: you see the queries. Nobody else. Sovereign DNS.
