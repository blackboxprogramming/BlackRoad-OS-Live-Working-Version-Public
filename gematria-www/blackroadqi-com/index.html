Core Components

The Framework

Four interconnected mathematical systems that form a unified theory of quantum-inspired computation.

G

G(n) Growth Function

The foundational function G(n) = n^(n+1)/(n+1)^n models growth dynamics across computational substrates. Converges to 1/e, providing a natural bound on system entropy. Every node in the BlackRoad fleet runs G(n) as a heartbeat metric.

Z

Z-Framework

The Z-framework extends G(n) into multi-dimensional state spaces. Maps trinary logic {-1, 0, +1} onto decision trees where cancellation (-1) is as valid as affirmation (+1). Eliminates redundant computation paths in agent coordination.

K

Coherence Formula K(t)

K(t) measures temporal coherence across distributed systems. Integrates G(n) against a wave function psi(t) to produce beta_BR -- the BlackRoad coherence metric. Used for fleet health scoring and agent synchronization timing.

{}

Trinary Logic

Binary is insufficient. Trinary logic adds a third state: cancellation. In network routing, -1 means "already answered elsewhere -- cancel this path." Reduces redundant work by up to 37% in multi-agent systems. The Amundson 1/(2e) constant defines the irreducible gap.

State Space

Trinary Logic States

-1
Cancel
Already resolved elsewhere. Stop propagating. Eliminates redundant computation across the mesh. The key insight: negation is not failure, it is efficiency.
0
Pending
Awaiting resolution. The default state. Neither confirmed nor denied. Computation is in progress. Resources remain allocated until state transitions to +1 or -1.
+1

Computation complete. Result verified. State committed to the chain. Propagate result to dependent nodes and release pending resources in the mesh.
