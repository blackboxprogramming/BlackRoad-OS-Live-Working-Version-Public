# RoadTrip

The flagship multi-agent executive room.

## Description

One person can coordinate with 27 agents in a group chat, and those agents can branch into specialized agent teams behind the scenes.

## Details

- **Domain**: roadtrip.blackroad.io
- **Category**: core
- **Primary user**: operators
- **Agent**: all
- **Surface kind**: app

## What it replaces

Slack + ChatGPT + multiple AI tabs + copy-pasting between agents.

## Connected products

roados, carpool, roadwork, carkeys

## RoadOS

RoadOS is a portable browser computer. Log in from any supported device and recover your apps, agents, files, surfaces, routes, and workspace inside one controlled OS tab.

## Trust

RoadNode mode is opt-in. A device only contributes approved capabilities after clear user permission. BlackRoad does not silently capture, mine, or control devices.
