## Purpose

Replaces fragmented, rented, frustrating computers with a single, device-agnostic highway where messy creativity is welcomed and boring work disappears. No installs. No fragmentation. One persistent, secure tab that remembers every mile.

## MVP Scope

- Full offline-first queuing with RoadChain sync
- CarKeys encryption at OS level
- Convoy Mode for group sessions
- Open Highway / Night Drive theme switcher

## Scale Mechanism

Any screen with a browser becomes a personal BlackRoad terminal. Users fork their own instance, agents clone per user, enabling 30T parallel sovereign highways.
