# CarKeys

The access and identity layer.

## Description

Instead of API keys everywhere, users get CarKeys: controlled identity, access, authentication, permissions, and signed routes.

## Details

- **Domain**: carkeys.blackroad.io
- **Category**: infrastructure
- **Primary user**: everyone
- **Agent**: silas
- **Surface kind**: service

## What it replaces

Scattered API keys, OAuth flows, password managers, identity fragmentation.

## Connected products

roados, roadchain, roadwork, roadtrip

## RoadOS

RoadOS is a portable browser computer. Log in from any supported device and recover your apps, agents, files, surfaces, routes, and workspace inside one controlled OS tab.

## Trust

RoadNode mode is opt-in. A device only contributes approved capabilities after clear user permission. BlackRoad does not silently capture, mine, or control devices.
